# 塔防游戏

基于 Phaser 3 + TypeScript 的网页塔防游戏。

## 运行

```bash
npm install
npm run dev
```

浏览器打开 http://localhost:5173

## 操作

1. 点击下方塔类型（炮塔 / 减速 / 溅射）选择要建造的塔
2. 点击地图上灰色网格非路径区域放置塔
3. 点击已放置的塔可升级（消耗 80 金币）
4. 守住 10 波敌人即胜利，生命归零则失败
5. 游戏结束后点击「点击重新开始」重玩

## 塔类型

- **炮塔** (100金): 单体伤害，优先攻击最接近终点的敌人
- **减速塔** (120金): 单体伤害 + 30% 减速 0.8 秒
- **溅射塔** (150金): 范围伤害

## 技术栈

- Phaser 3
- TypeScript
- Vite
