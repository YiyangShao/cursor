# 塔防小游戏 (React + TypeScript)

基于 React + TypeScript + Vite 的浏览器塔防游戏。

## 运行

```bash
npm install
npm run dev
```

访问 `http://localhost:5173`

## 塔类型

| 塔 | 造价 | 特点 |
|----|------|------|
| 🏹 箭塔 | 100 | 快速单点伤害 |
| 💣 炮塔 | 200 | 范围溅射伤害 |
| ❄️ 减速塔 | 150 | 减速敌人 + 轻伤 |
| 🔮 法师塔 | 180 | 高单点伤害 |

## 敌人类型

| 敌人 | 血量 | 速度 | 奖励 |
|------|------|------|------|
| 哥布林 | 低 | 中 | 15 |
| 狼 | 低 | 快 | 25 |
| 坦克 | 高 | 慢 | 40 |
| Boss | 极高 | 慢 | 150 |

## 技术栈

- React 19 + TypeScript
- Vite
- Canvas 2D 渲染
- Web Audio API 音效
- 粒子系统
